# Quality Bar

This document defines the quality and formatting standards for all contributions to ARMAGEDDON Test Suite.

1. **A-Grade Quality**: Must pass SonarCloud quality gates.
2. **Determinism**: Code behavior must be strictly deterministic. "If it is not deterministic, it is a bug."
3. **No Hedging**: Documentation and artifacts must use factual, evidence-backed language without hedging (e.g. "maybe", "probably").
